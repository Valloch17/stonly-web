Stonly Guide Builder

Create guides and steps from YAML with the Stonly API.

Guide defaults

These values pre-fill the payload. YAML values override them when provided.

Guide YAML

Describe the guide structure using nested steps and choices.

YAML format

  • Top-level guide object must include contentTitle, language, and firstStep.
  • Each step defines a title, content (HTML allowed), and optional choices array.
  • Choices accept label, optional position, and a nested step.
  • Optional step fields: media (up to 3 URLs), position, language.
